# Changelog

All notable changes to Apex A+ Academy are documented here.

## 1.1.0 - 2026-06-14

### Added

- Global `Ctrl+K` search across domains, objectives, questions, explanations, and flashcards
- Focused practice sessions launched directly from domain recommendations and learning paths
- Accurate daily question activity and study streak tracking
- Study-status notification panel for daily goals, due cards, and streaks
- Portable JSON progress backup and local data reset controls
- Flashcard queue completion and optional full-deck review mode

### Fixed

- Checked practice answers can no longer be changed after reviewing the explanation
- Existing notes now load their title and body safely when the notes view opens
- Flashcards no longer recycle immediately after the scheduled queue is complete
- Daily mission progress no longer uses lifetime question totals
- Mobile navigation starts closed and dismisses after choosing a destination
- Bookmark controls now toggle instead of only adding items

## 1.0.0 - 2026-06-13

- Initial Windows desktop release with learning paths, practice sessions, flashcards, analytics, notes, bookmarks, themes, local persistence, and NSIS packaging
