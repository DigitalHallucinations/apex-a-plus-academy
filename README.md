# Apex A+ Academy

An offline-first Windows desktop study environment for CompTIA A+ preparation. It is an independent educational product and is not affiliated with or endorsed by CompTIA.

## Development

```powershell
npm install
npm run desktop:dev
```

## Package

```powershell
npm run desktop:build
```

The NSIS installer is written under `src-tauri/target/release/bundle/nsis/`.
